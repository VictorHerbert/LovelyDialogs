# Lovely Dialogs

A flutter implementation of [LovelyDialog](https://github.com/yarolegovich/LovelyDialog)

## Import

In your pubspeck.yaml:

'''
dependencies:
  flutter:
    sdk: flutter

  lovelydialogs: 0.0.1
'''

and import

''' dart
import 'package:badges/badges.dart';
'''
